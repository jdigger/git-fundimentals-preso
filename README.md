

# Uses

* [reveal.js](https://travis-ci.org/hakimel/reveal.js), a framework for easily creating beautiful presentations using HTML.
* [SASS](http://sass-lang.com/) for styling
* [graphviz](http://graphviz.org/) for creating charts
